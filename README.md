# 📊 Amazon Sales Analysis Dashboard

## 🔍 Overview
This project analyzes Amazon sales data using SQL and Power BI.

## 🔄 Project Workflow
1. Data collected in CSV format  
2. Data cleaned and transformed using SQL  
3. KPIs calculated (Revenue, Orders, AOV, Cancellation %)  
4. Dashboard built in Power BI  
5. Insights generated for business decision making  

## 📌 Key KPIs
* Total Revenue: 78.59M  
* Total Orders: 120K  
* Total Quantity: 117K  
* AOV: 652.88  
* Cancellation Rate: 14.21%

## 📊 Key Insights

- Highest revenue generated in April (~21M)
- Kurta category contributes maximum sales
- Amazon fulfillment dominates over merchant fulfillment
- Cancellation rate is 14.21%, indicating potential operational issues
- Sales drop significantly after June
